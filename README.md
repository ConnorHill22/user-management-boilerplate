# User Management Boiler Plate

Basic boiler plate repo for a basic user management system using Supabase, Golang, htmx, and a sprinkle of typescript

## Technology
- Databse (supabase)
- Authentication (supabase)
- Backend (Golang [Echo])
- Frontend
  - HTMX
  - Server components
  - Tailwind
  - Some other sprinkles of TS
